
# ✅ React Todo List App

This project is a **simple Todo List application built using React**.  
It allows users to create, manage, and remove daily tasks through an interactive user interface.

The application demonstrates core **React concepts such as components, state management, and user interaction** while building a practical productivity tool.

---

# 🚀 Features

- ➕ Add new tasks
- 📋 View all tasks in a list
- ✔️ Mark tasks as completed
- ❌ Delete tasks
- ⚡ Fast and responsive interface

---

# 🛠 Tech Stack

This project uses modern front-end technologies:

- **React.js** – UI framework
- **JavaScript (ES6+)** – Application logic
- **HTML5** – Structure
- **CSS3** – Styling and layout

# ⚙️ How the Application Works

1. The application loads a **React interface** displaying a task input field.
2. Users can **enter a task and add it to the list**.
3. Each task is rendered dynamically using **React components**.
4. Tasks can be **marked as completed or removed from the list**.
5. React updates the UI automatically whenever the state changes.

# ▶️ Run the Project Locally

### 1️⃣ Clone the repository

```
git clone https://github.com/nischalare/ReactTodoListApp.github.io.git
```

### 2️⃣ Navigate to the project folder

```
cd ReactTodoListApp.github.io
```

### 3️⃣ Install dependencies

```
npm install
```

### 4️⃣ Start the development server

```
npm start
```

The application will open in your browser at:

```
http://localhost:3000
```
